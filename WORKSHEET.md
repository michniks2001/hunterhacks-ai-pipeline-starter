## 1. User Input

**What does the user give your system?**

Examples: a question, topic, file, image, URL, location, preferences, form fields.



---

## 2. Context / Data

**What extra information does your system give the model?**

Examples: documents, search results, database records, user preferences, examples, instructions, uploaded files.



---

## 3. Tools / APIs

**What external tools or APIs does your system use?**

Examples: Groq, OpenAI, Gemini, database, search API, maps API, ChromaDB, LangChain, LlamaIndex, webhooks.



---

## 4. Model

**What should the AI model do with the input and context?**

Examples: summarize, classify, recommend, answer, generate JSON, create a plan, transform data.



---

## 5. Output

**What does your system return to the user?**

Examples: text, markdown, recommendations, JSON, UI theme, chart, image, code, file.



---

## 6. Check / Eval

**How will you know the output is useful, correct, or good enough?**

Examples: follows the format, uses the context, avoids fake facts, is specific, displays correctly in the app.



---
